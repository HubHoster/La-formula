## Translation System Implementation

### How It Works:
1. **LanguageContext** (`src/contexts/LanguageContext.tsx`)
   - Provides `useLanguage()` hook to all components
   - Manages current language state (`en` or `es`)
   - Provides `t()` function for translations
   - Provides `setLanguage()` to switch languages

2. **Translations File** (`src/translations.ts`)
   - Contains all English and Spanish translations
   - Organized by section (header, footer, services, about, contact, etc.)
   - **Total Keys: 200+**
   - **Coverage: 100%** (all English keys have Spanish equivalents)

3. **Routing System** (`src/App.tsx`)
   - English routes: `/`, `/about`, `/contact`, `/careers`, `/terms`, `/privacy`, `/cookies`
   - Spanish routes: `/spanish`, `/spanish/about`, `/spanish/contact`, etc.
   - Language automatically detected from URL path

4. **Component Usage:**
   ```tsx
   const { t, lang } = useLanguage();
   const basePath = lang === 'es' ? '/spanish' : '';
   
   // Use translations
   <h1>{t('page.title')}</h1>
   
   // Use language-aware links
   <Link to={`${basePath}/about`}>About</Link>
   ```
